# Gerador de Recompensas para Império de Jade RPG

**Utilize esse website para gerar as recompensas para a sua campanhade RPG baseado apenas no ND, sem precisar rolar dados!**

## Sobre

Adentre o mundo de aventuras e intrigas do Império de Jade, onde cada missão enfrentada reserva uma recompensa única e emocionante. Escolha o ND do inimigo derrotado e o algoritmo rolará os dados para você!

## Como funciona

Para a criação de recompensas completamente novas e aleatórias, basta selecionar o Nível de Dificuldade na caixa `seletor de ND` e tudo será gerado de acordo com o ND escolhido.
